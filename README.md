📊 Interactive Business Dashboard (Streamlit)
🎯 Task Objective

The objective of this project is to develop an interactive business dashboard using the Global Superstore Dataset.
The dashboard helps analyze sales, profit, and customer performance using dynamic filters and visual KPIs.

🛠️ Approach
Loaded the Global Superstore dataset using Pandas
Cleaned the data by:
Removing duplicates
Handling missing values
Converting date columns into datetime format
Built an interactive dashboard using Streamlit
Added sidebar filters for:
Region
Category
Sub-Category
Created key performance indicators (KPIs):
Total Sales
Total Profit
Top 5 Customers by Sales
Used Plotly for interactive visualizations
📈 Results and Findings
The dashboard allows users to filter data dynamically by region, category, and sub-category
Sales and profit KPIs update instantly based on selected filters
The analysis shows:
Certain categories generate higher sales but lower profit
A small group of customers contributes to the majority of sales
Regional performance varies significantly in profitability
The dashboard improves decision-making by providing a clear visual understanding of business performance# streamlit-business-dashboard
